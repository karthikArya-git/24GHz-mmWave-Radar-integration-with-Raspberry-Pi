# 24GHz mmWave Radar Integration with Raspberry Pi Pico

## Overview
This project demonstrates real-time human motion detection using the RD-03D 24GHz mmWave Radar Sensor and Raspberry Pi Pico.

The radar detects human presence and motion, while the Raspberry Pi Pico processes the radar data and sends it to a PC for real-time visualization.

## Hardware Used
- RD-03D 24GHz mmWave Radar
- Raspberry Pi Pico
- USB Cable
- PC/Laptop

## Software Used
- MicroPython
- Processing IDE
- Thonny IDE

## Features
- Human presence detection
- Motion tracking
- Real-time visualization
- Low power consumption
- UART communication
